## О проекте

**BA Workspace Pro** — одностраничное веб-приложение для бизнес- и системных аналитиков. Позволяет структурированно работать над задачами, используя специализированные шаблоны вопросов для различных типов задач, отслеживать время, вести аналитику и интегрироваться с Jira, Confluence и Telegram.

## Возможности

- **10 типов задач** — каждый со своим набором вопросов-шаблонов (16–23 вопроса на тип)
- **Структурированные секции** — вопросы и ответы, критерии приёмки, решения, риски, хвосты (кто/что/срок), зависимости, заметки
- **Таймер** — учёт времени по каждой задаче с цветовой индикацией
- **AI-ассистент** — предложение вопросов, суммаризация, анализ рисков, генерация критериев
- **Аналитика** — сводка по задачам, распределение по типам/статусам/приоритетам, активность
- **Экспорт в Markdown** — выгрузка любой задачи
- **Тёмная тема** — полная поддержка dark mode
- **Интеграции** — подготовлены API-роуты для Jira, Confluence и Telegram
- **Локализация** — русский и английский языки
- **Локальное хранение** — все данные сохраняются в localStorage

## Установка и запуск

### Предварительные требования

- **Node.js** 18+ (рекомендуется 20+)
- **bun** (рекомендуется) или npm/yarn/pnpm

### Установка

```bash
git clone <repo-url>
cd ba-workspace-pro
bun install
bun run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000).

### Сборка для продакшена

```bash
bun run build
bun run start
```

## Конфигурация

### База данных

По умолчанию используется SQLite (файл `prisma/dev.db`). Настройте через `DATABASE_URL` в `.env`:

```bash
bun run db:generate  # Сгенерировать Prisma client
bun run db:push      # Применить схему к БД
bun run db:migrate   # Создать и применить миграцию
```

### Интеграции

Настройки Jira, Confluence и Telegram задаются через интерфейс Settings в приложении.
